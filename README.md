# Rspress Theme AIm Template

> A lightweight template for Rspress documentation sites.

## 🚀 Quick Start

### 1. Use this template

Click "Use this template" to create your repository.

### 2. Update configuration

Update `rspress.config.ts`:
```typescript
base: '/your-repo-name/',
title: 'Your Project Title',
```

### 3. Enable GitHub Pages

Settings → Pages → Source: GitHub Actions

### 4. Deploy

Push your changes and wait for the deployment.
